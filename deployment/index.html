EM-FCEUX

em-fceux is an Emscripten port of FCEUX.

Source code: https://bitbucket.org/tsone/em-fceux/

HOW TO PLAY

To play a game, drag and drop it (the .nes file or similar) to this browser window.

Played games are added to the game stack on the right. You can click a game on the stack to play it.

To remove a game, click on the 'X' icon. (The built-in game '2048' cannot be removed.)

Supported file formats: .nes, .nsf, .zip

SETTINGS TAB

Picture and emulation settings are on the settings tab on the left (click on the pixelated triangle).

The settings can for example toggle composite NTSC and TV emulation and adjust colors.

For maximum frame rate, disable the TV emulation and use full-screen mode.

KEYS

Game controller:

Arrow keysD-pad directions
FA button
DB button
SSelect button
EnterStart button

Other:

Ctrl-RReset system
EscExit full-screen mode
TabSpeed throttle
0-9Select save state
F5Save save state
F7Load save state
PPause emulation
\Single frame step

Only one game controller is supported. Keyboard input customization is not supported.

TECHNICAL

Game and save data are stored locally in the user's web browser. This data is never transmitted out from the web browser.

Extracting game or save data from the local browser storage is not supported.

Please refer to the source code project page for more information.
